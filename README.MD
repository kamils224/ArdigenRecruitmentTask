# Full-stack Developer Recruitment Task 

## Project setup (Fast API Backend)
Create python virtual environment and from the root directory run:

### Install dependencies
```
pip install -r requirements.txt
```
### Run Fast API server
```
uvicorn main:app 
```
### Format code
```
black .
```
